## 4. Habit Loop Design

### Daily Loop (Agent Surfaces)
```
User opens Claude Code / OpenClaw
  → starts workflow via TA MCP tools
  → TA captures trajectory
  → TA replays saved path
  → TA shows savings
  → User sees value daily
```

### Weekly Loop (Dashboard)
```
User opens retention.sh dashboard
  → sees weekly savings summary
  → reviews optimization candidates
  → approves verified shortcuts

  → shares report to team/Slack
```

### Monthly Loop (Business)
```
Team reviews retention.sh portfolio
  → sees cumulative savings
  → reviews workflow durability

  → renews/expands subscription
```

## 5. Downstream Data Strategy

### What We Are NOT
We are NOT a data brokerage company. We do not sell user data or workflow traces.

### What We Can Offer (Second-Order)
In select cases, with explicit permission and proper anonymization:

| Data Type | Potential Use | Partner Type |
|-----------|-------------|--------------|
| Anonymized benchmark data | Agent evaluation datasets | Research labs |
| Synthetic workflow templates | Training data augmentation | Model companies |
| Aggregate savings statistics | Industry benchmarks | Analyst firms |
| TCWP spec adoption | Schema standardization | Ecosystem partners |

### Rules:
- Data partnerships are second-order, not primary business
- Always require explicit customer consent
- Always anonymize or synthesize before sharing
- Never compromise customer trust for data revenue
- The brand is workflow intelligence, not data exhaust

## 6. Competitive Positioning

### Against BrowserStack / Device Clouds
> We are not replacing your device cloud. We are the intelligence layer above it. Use BrowserStack for execution breadth; use TA for verification memory and replay economics.

### Against Generic AI Testing (Momentic, etc.)
> Generic AI testing runs every workflow from scratch every time. TA captures the trajectory, compresses it, and replays the cheapest valid path. Each rerun gets cheaper.

### Against Manual QA
> Manual QA is expensive, slow, and non-repeatable. TA produces structured, auditable artifacts with deterministic replay and visible savings.

### Against Agent Runtimes (Browser Use, etc.)
> Agent runtimes execute. TA remembers, replays, audits, and compresses. They are complementary, not competing.
