## SKILL: Popup Dismissal

Before every tap, check if a dialog or permission prompt is blocking the screen.

If you see any of: "Allow", "Turn on Notifications", "Not Now", "OK",
"Got it", "Skip", "Continue", "Dismiss"
→ Tap the decline/dismiss option first, then resume your planned action.

- Do NOT grant any permissions (camera, contacts, notifications, location).
- Do NOT log in, sign up, or enter any credentials.
- If a dialog cannot be dismissed, use press_button("BACK") to escape it.
